# Body Performance Analysis Project

## Project Overview
This project analyzes body performance data and builds a Neural Network model to classify individuals based on physical attributes.

---

## Dataset
The dataset includes:
- Age
- Gender
- Height & Weight
- Body Fat Percentage
- Blood Pressure
- Grip Strength
- Physical Performance Tests

---

## Data Processing
- Removed duplicates  
- Handled unrealistic values  
- Dropped missing data  
- Converted categorical variables  
- Feature scaling applied  
- Created age groups  

---

## Model Used
- Neural Network (MLP) using TensorFlow/Keras  
- 2 Hidden Layers  
- Dropout to reduce overfitting  
- Softmax activation for multiclass classification  

---
